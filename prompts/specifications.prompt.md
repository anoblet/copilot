- Analyze the entire project
- Generate a comprehensive specification document for the project
- Place the document in (SPECIFICATION.md).
- Submodules are considered separate packages

Use the following format:

```markdown
# ${packageName}

## Specifications

- [x] Specification 1
  - [] Specification 1-1
    - [] ...
  - [] Specification 1-2
- [x] Specification 2
  - [] Specification 2-1
    - [] ...
  - [] Specification 2-2
```
